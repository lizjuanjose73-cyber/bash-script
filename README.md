# Linux System Monitor (Bash)

A simple system monitoring dashboard built with Bash scripting.

## Features

- Monitor CPU usage
- Monitor RAM usage
- View top 10 active processes
- Check disk space
- Interactive terminal menu

## Technologies Used

- Bash scripting
- Linux system commands
- top
- free
- ps
- df

## How to Run

Clone the repository:

git clone https://github.com/lizjuanjose73-cyber/bash-script.git

Go to the directory:

cd bash-script

Make the script executable:

chmod +x sistema.sh

Run the script:

./sistema.sh
